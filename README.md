# Weather App

A simple Python weather application that fetches live weather data using the OpenWeatherMap API and displays the current conditions

## Features

- Displays current temperature
- Shows weather conditions and description
- Auto refreshes weather data
- Displays humidity, pressure, and wind speed

## Technologies Used

- Python
- Tkinter
- Requests
- OpenWeatherMap API

## Future Improvements

- 5 day forecast
- Improved user interface
- Raspberry Pi weather station integration
- Supprt for multiple cities
